Problem Statement

The sinking of the Titanic is one of the most infamous shipwrecks in history. On April 15, 1912, during her maiden voyage, the widely considered “unsinkable” RMS Titanic sank after colliding with an iceberg. Unfortunately, there weren’t enough lifeboats for everyone onboard, resulting in the death of 1502 out of 2224 passengers and crew.

While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others. Some of the predictor variables include the age, sex, fare, cabin of the passengers.

The outcome of this predictive modelling is to predict a passenger's likelihood of survival in the titanic shipwreck

def ApplyStratifiedKFold(train_features, train_targets, model):
    '''
    function to apply Statified KFold cross validation 
        
    args:
        train_features: independent feature data to split into n_folds.
        
        train_targets: dependent target data to split into n_folds.
        
        model: Machine learning model to fit data on.
        
    return:
        eval_dfs: a list of pandas datafames that contains metric results for each n_fold.
                
    '''
    kfold=StratifiedKFold(n_splits=n_folds)
    index=1
    eval_dfs = []
    for train_index, test_index in kfold.split(train_features, train_targets):
        print('Number {}, Fold'.format(index))
        X_train, X_val = train_features.iloc[train_index], train_features.iloc[test_index]
        y_train, y_val = train_targets.iloc[train_index], train_targets.iloc[test_index]
        
        curr_model = FitModel(model, X_train, y_train)  
        eval_df = EvaluateModel(curr_model, X_val, y_val)
       
        eval_dfs.append(eval_df)
        index+=1

    return eval_dfs

        y: Training set target data (dependent feature)

        model: A fitted model on the dataset
    '''

def EvaluateModel(model, test_features, test_targets, threshold=0.5):    
    '''
    function to evaluate the performance of a model on the test data
    
    args:
        model: The model to evaluate
        
        test_features: the independent feature values of the test data to be evaluated
        
        test_targets: the dependent target values of the test data to be evaluated
        
        threshold: threshold to apply on predictions in creating categorical values.
    
    returns:
        eval_df: A pandas datafame that contains metric results 
    '''
    
    eval_df = pd.DataFrame()
    cont_preds = model.predict_proba(test_features)[:,1]
    cat_preds = pd.Series(cont_preds).apply(lambda x: 0 if x <= threshold else 1)
    
    eval_df['Accuracy_score'] = pd.Series(accuracy_score(test_targets, cat_preds), name='Accuracy')
    eval_df['F-Measure_score'] = pd.Series(f1_score(test_targets, cat_preds), name='f1_score')
    eval_df['AUC_score'] = pd.Series(roc_auc_score(test_targets, cont_preds), name = 'AUC')
    
    
    return eval_df

def find_sem(metric, model, return_err=False):
    '''
    args:
        metric: metric data to find standard error of the mean (Accuracy, F-Measure, AUC)
        
        model: KNN or RFC 
        
        return_err: whether to return standard error value or not (True, False)
    '''
    
    if model.lower() == 'knn':
        err = sem(knn_eval_df['{}_score'.format(metric)])
        print('The standard error of mean of {} score for the {}-classifier = {}'.format(metric, model.upper(), err))
    
    elif model.lower() == 'rfc':
        err = sem(rfc_eval_df['{}_score'.format(metric)])
        print('The standard error of mean of {} score for the {}-classifier = {}'.format(metric, model.upper(), err))
    
    if return_err:
        return err

Accuracy is the degree to which the information correctly describes the phenomena it was designed to measure.

Accuracy is the fraction of predictions our model got right : Accuracy = Number of correct predictions / Total number of predictions.

For this use-case, Accuracy is the fraction of passengers, our model was able to predict survuved the titanic shipwreck.

PAIRED SAMPLE T-TEST ASSUMPTIONS¶

In order for the paired sample t-test results to be trusted, the following assumptions need to be met:

The observations are independent.

The Dependent Variable should be approximately normally distributed. The paired sample t-test is very sensitive to this violation. when there is a violation of normality, if it’s not a major violation then the test results can be considered valid.

The Dependent Variable should not contain any significant outliers.

The histograms of our data does not seem to be completely normally distributed.

The distribution for the K-Neighbors classifier seems a little skewed to the right, while the distribution of the Random forest classifier seems to be normally distributed but with a tiny skew to the left.

Let’s test this statistically to see if the data is normally distributed.

To test this, I use the Shapiro-Wilk test for normality, The output is not labeled, however the first value is the W test value, and the second value is the p-value.

Therefore I reject the null hypothesis in support of the alternative.
